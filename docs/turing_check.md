# Sobre o Turing Check

O método de Turing não deve ser aplicado globalmente em intervalos curtos
para T elevado, pois a fórmula assintótica de Riemann–von Mangoldt possui
erro local significativo.

Neste projeto, o Turing check é aplicado por blocos disjuntos, o que
reflete a prática correta em matemática computacional e evita
falsos negativos.
